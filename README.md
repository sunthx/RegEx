# regular_expression_store

> 总结正则表达式在开发时常用的场景及收集一些常用的正则表达式。

## Email

<p>[\w!#$%&'*+/=?^_`{|}~-]+(?:\.[\w!#$%&'*+/=?^_`{|}~-]+)*@(?:[\w](?:[\w-]*[\w])?\.)+[\w](?:[\w-]*[\w])?</p>

## 网址

[a-zA-z]+://[^\s]*

## 身份证（18位）

^(\d{6})(\d{4})(\d{2})(\d{2})(\d{3})([0-9]|X)$

## 中文字符

[\u4e00-\u9fa5]

## 空白行

\n\s*\r

## 中国邮政编码

[1-9]\d{5}(?!\d)
